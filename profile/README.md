## 📆 스터디 진행 방식

1. **문제 선정**  
   - 매주 특정 주제(ex: DP, BFS/DFS 등)에 맞춰 알고리즘 문제 선정  
   - 난이도: **Easy ~ Hard** (균형 잡힌 문제 배분)
   - 매주 월, 수, 금, 토요일마다 문제 풀고 노션에 문제 정리
   
2. **코드 작성 및 리뷰**  
   - 각자 GitHub에 코드 업로드  
   - 다른 멤버의 코드를 리뷰하며 개선점 논의  

3. **정리 및 발표**  
   - 매주 화요일, 랜덤으로 하나의 문제를 골라 발표 진행
   - 효과적인 접근법, 최적화 방법 논의
